# eldee Brand Book

Profesionální brandbook značky eldee — fotbalové štulpny s dvěma chráněnými průmyslovými vzory.

🌐 **Live:** https://brand.eldee.cz (po DNS) | https://eldee-brandbook.vercel.app

📄 **PDF:** https://brand.eldee.cz/eldee-brandbook.pdf

## Development

```bash
npm install              # Install dependencies
npm run dev              # Local server at :4321
npm run build            # Production build
npm run preview          # Preview production build locally
npm run generate-pdf     # Generate /public/eldee-brandbook.pdf
npm test                 # Unit + E2E tests
```

## Stack

- Astro 6.x (statický SSG)
- Tailwind CSS 4 (CSS @theme design tokens)
- TypeScript (komponenty)
- Playwright (E2E + PDF generation)
- Vercel (hosting + CDN)
