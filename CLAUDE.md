# eldee brand book (eldee-brandbook.vercel.app)

VEŘEJNÉ repo + web (Astro: `npm run build` → `dist/`, deploy Vercel). Push = web se aktualizuje. Žádná citlivá data. Mluv česky, tykej, laicky. Mobile-first.

Obsah: `src/data/content.json` + `src/pages/*.astro`.

## Úkoly + stav (Hledík i Doležal)
Úkoly / milníky / stav projektu se NEvedou tady — vedou se v `eldee-hq/data/stav.json` (HQ nástěnka), příběh v `eldee-business/aktualni-stav.md`. Když při práci na brandbooku vznikne úkol, zapiš ho do `stav.json` v eldee-hq (stav `další`, `kdo`, `vzniklo`). Plný postup: `eldee-business/CLAUDE.md` → „POVINNÝ WORKFLOW".
